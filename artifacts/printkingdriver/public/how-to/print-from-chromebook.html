How to Print from a Chromebook

Google Cloud Print was retired in 2020. Modern Chromebooks use direct IPP printing, which works with most network printers.

Quick Steps

  • Confirm the printer is IPP-Everywhere certified — Mopria-certified printers all support IPP-Everywhere. Most modern devices qualify.
  • Open Chromebook print settings — Settings » Advanced » Printing » Printers.
  • Add the printer — Auto-discovery finds nearby IPP printers. Pick yours and click Save.
  • Add manually if needed — Choose "Add manually" and enter the printer's IP and IPP path (usually /ipp/print).
  • Print a test page — Open any document, choose Print, pick the printer, and confirm it works before relying on it.

Why This Works

The sequence above is the order the operating system actually expects. Skipping a step (running the installer without admin rights, not printing the test page, not rebooting) is what causes the "this should have worked but didn't" outcomes that send people in circles.

When to Stop and Try Something Else

If you have run through every step above and the issue persists, the problem is almost certainly not the procedure — it is the underlying configuration. Common second-line causes:

  • Driver mismatch — the model on the box is not the model in the driver download. Confirm using the label on the printer itself, not the box.
  • Permissions — non-admin user accounts often cannot install drivers or modify print queues. Try the same steps from an admin account.
  • Antivirus blocking installer files — temporarily pause your security software during install.
  • Vendor server outage — vendor support sites occasionally go down for hours. Try again later or check the vendor status page.

Why It Matters

Each of these "small" tasks is a daily friction point that compounds over time. Five minutes lost per print job, ten times a day, costs you several hours a month. Doing it right once and saving the procedure is worth the upfront effort.
