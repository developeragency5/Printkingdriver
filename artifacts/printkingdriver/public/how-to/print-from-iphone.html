How-To Guide

How to Print from an iPhone or iPad

AirPrint is built into iOS. Almost every device made in the last decade supports it.

Quick Summary

Printing from an iPhone uses AirPrint — Apple's built-in driverless printing protocol — to send jobs to any AirPrint-certified device on the same Wi-Fi network. No app or driver download is required. The share button in any app reveals the Print option if a compatible device is detected.

  • The iPhone and device must be on the same Wi-Fi network — mobile data will not work
  • Tap the Share button (box with arrow) in any app, then tap Print
  • If no devices appear, check the device is AirPrint certified and powered on

Quick Steps

  • Confirm the device is AirPrint-certified — Look on the box, the device panel or the vendor support page. Most modern devices are.
  • Connect both to the same Wi-Fi — AirPrint uses local mDNS discovery — both devices must be on the same network.
  • Open the document or photo — Tap the Share button, then scroll down to Print.
  • Pick the device — iOS shows every AirPrint device on the network. Pick yours.
  • Adjust options — Tap the device name to change paper size, copies, double-sided and page range.

Why This Works

The sequence above is the order the operating system actually expects. Skipping a step (running the setup tool without admin rights, not printing the test page, not rebooting) is what causes the "this should have worked but didn't" outcomes that send people in circles.

When to Stop and Try Something Else

If you have run through every step above and the issue persists, the problem is almost certainly not the procedure — it is the underlying configuration. Common second-line causes:

  • Driver mismatch — the model on the box is not the model in the driver download. Confirm using the label on the device itself, not the box.
  • Permissions — non-admin user accounts often cannot set up drivers or modify print queues. Try the same steps from an admin account.
  • Antivirus blocking setup tool files — temporarily pause your security software during set up.
  • Vendor server outage — vendor support sites occasionally go down for hours. Try again later or check the vendor status page.

Why It Matters

Each of these "small" tasks is a daily friction point that compounds over time. Five minutes lost per print job, ten times a day, costs you several hours a month. Doing it right once and saving the procedure is worth the upfront effort.
