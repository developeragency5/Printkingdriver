Printer & Driver Glossary

Plain-English definitions for the printer, driver and print-stack terms used across this site and in vendor documentation.

How to Use This Glossary

The terms below come up regularly when reading driver download pages, vendor manuals and our own troubleshooting guides. Each definition is written for the person who has just hit the term while trying to fix a printer, not for the engineer who designed the standard. Where a term has a different name on macOS and Windows, both names are listed.

A–Z Glossary

TermMeaning
AirPrintApple's implementation of IPP that lets iPhones, iPads and Macs print to compatible printers without installing a driver.
CalibrationA self-test routine that adjusts colour or alignment to match a reference. Recommended after replacing a drum, toner or print head.
CUPSCommon Unix Printing System — the print queue manager used on macOS and most Linux distributions. Roughly equivalent to the Windows Print Spooler.
DHCPDynamic Host Configuration Protocol — assigns IP addresses to devices on a network. If a printer's DHCP-assigned address changes, the print queue may stop working until the port is updated.
DPIDots Per Inch — the resolution of a print, expressed as the number of ink or toner dots per linear inch. Higher DPI means finer detail but slower printing and more ink/toner used.
DriverA small piece of software that lets the operating system talk to a piece of hardware. Without the right driver, the operating system can see that something is plugged in but not what it is or how to use it.
DrumThe photosensitive cylinder in a laser printer that picks up toner and transfers it to the page. Drums wear out over thousands of pages and may need replacing separately from the toner.
Duplex PrintingPrinting on both sides of the page. Some printers do this automatically (auto-duplex); others require the user to flip the stack.
FirmwareSoftware stored permanently inside a piece of hardware (a printer, motherboard, router). Firmware is updated less often than drivers and usually only when a security or compatibility fix is released.
FuserThe hot roller assembly in a laser printer that bonds toner to the page. The fuser stays hot for several minutes after power-down — never reach inside immediately after switching off.
ICAImage Capture Architecture — the macOS equivalent of TWAIN, built into the operating system rather than installed per-device.
IPPInternet Printing Protocol — a modern, vendor-neutral protocol that lets a computer print to a network printer without installing a vendor driver. The basis of AirPrint and Mopria.
MopriaThe Android equivalent of AirPrint, also based on IPP. Supported by most modern network printers regardless of manufacturer.
Nozzle CheckA built-in test page on inkjet printers that prints a pattern from each ink nozzle. Missing lines indicate clogged nozzles that need a cleaning cycle.
PCLPrinter Command Language — a page-description language developed by HP and supported by most office printers. Generally faster than PostScript for plain office documents.
PostScriptA page-description language developed by Adobe, widely used in graphics and design printing. Slower than PCL for plain text, more accurate for complex graphics.
Print QueueThe list of jobs waiting to be sent to a printer. A queue is created automatically when you add a printer and is managed by the print spooler.
Print ServerA computer (or dedicated appliance) that hosts shared print queues for a network. Other computers send jobs to the server, which forwards them to the right printer.
Print SpoolerThe Windows service that holds print jobs in a queue and sends them to the printer one at a time. The macOS equivalent is CUPS. When the spooler hangs, the print queue freezes.
SANEScanner Access Now Easy — the open-source scanner interface used on Linux. Roughly equivalent to TWAIN on Windows.
SNMPSimple Network Management Protocol — used by Windows and management tools to check whether a network printer is reachable. A failed SNMP check is the most common reason a printer is reported "offline".
Static IPA network address that is fixed rather than assigned by DHCP. Recommended for office printers so the print queue does not break when the address changes.
TonerThe dry powder used in laser printers, fused to the page by heat. Equivalent to ink in inkjets.
TWAINThe standard interface for scanners on Windows. A scanning application talks to the scanner through TWAIN; the TWAIN driver is supplied by the scanner manufacturer.
Universal DriverA single driver that works across many printer models from the same manufacturer. Used on Windows print servers to simplify fleet management.

Still Stuck on a Term?

If you read something in a vendor manual or on a download page that is not covered here, contact us and we will add a definition. The glossary is intended to grow with the questions readers actually ask.
