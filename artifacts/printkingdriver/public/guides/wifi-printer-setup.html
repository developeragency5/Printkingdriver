Setup Guide

Wi-Fi Printer Setup

Everything that has to line up for a wi-fi printer setup to work first time, and what to do when one of those things does not.

Overview

Adding a printer over Wi-Fi is the most flexible way to share it across a household or office, but it is also where the most setup problems happen. The printer, the router and the computer all have to agree on the same network and security settings before the connection works.

Step-by-Step

  • Confirm the network — Both the printer and the computer need to be on the same Wi-Fi SSID. Many routers split 2.4 GHz and 5 GHz networks; printers usually only support 2.4 GHz, so connect the computer to the 2.4 GHz network temporarily during setup.
  • Run the printer wireless wizard — On the printer's front panel, navigate to Network or Wireless Setup and choose the Wi-Fi setup wizard. Pick your SSID from the list and enter the password using the panel keypad.
  • Use WPS for one-touch setup (optional) — If the router has a WPS button, press it and within two minutes press the WPS button on the printer. The two devices will negotiate the connection automatically.
  • Add the printer on the computer — Once the printer reports a successful connection, follow your operating system's standard add-printer steps to discover it on the network.
  • Set a DHCP reservation — In the router admin page, find the printer's MAC address and reserve its IP. This stops the IP changing on every reboot, which is a frequent cause of "printer offline" errors.

What Can Go Wrong

A short list of the issues people most often run into during wi-fi printer setup, and the right fix for each:

SymptomLikely CauseFirst Step
Connection succeeds but no printDriver mismatched to the model.Reinstall the vendor driver.
Connection drops after a few minutesPower-saving or sleep settings.Disable sleep on the printer or computer.
Printer not discoveredDiscovery protocol blocked or different subnet.Confirm both devices on same network.
