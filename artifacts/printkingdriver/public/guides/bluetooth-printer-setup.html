Setup Guide

Bluetooth Printer Setup

Everything that has to line up for a bluetooth printer setup to work first time, and what to do when one of those things does not.

Overview

Bluetooth printing is most common on portable receipt and label printers, and a few mobile-targeted inkjet models. It is slow and short-range compared to Wi-Fi, but it requires no network at all, which makes it useful for ad-hoc and field use.

Step-by-Step

  • Pair the printer — On the printer, enable Bluetooth pairing mode. On the computer or phone, choose Add Bluetooth Device and pick the printer from the list.
  • Enter the PIN if prompted — Many Bluetooth printers have a default PIN of 0000 or 1234. Check the printer manual.
  • Install the driver — For one-off paired use the OS may install a generic driver. For regular use install the vendor driver, which adds support for the printer's actual paper sizes.
  • Confirm the range — Bluetooth is reliable up to about 10 metres line-of-sight. Walls and metal cabinets reduce the range significantly.
  • Repair if dropped — If the connection drops, remove the printer and re-pair from scratch. Bluetooth pairing data sometimes gets corrupted after a printer firmware update.

What Can Go Wrong

A short list of the issues people most often run into during bluetooth printer setup, and the right fix for each:

SymptomLikely CauseFirst Step
Connection succeeds but no printDriver mismatched to the model.Reinstall the vendor driver.
Connection drops after a few minutesPower-saving or sleep settings.Disable sleep on the printer or computer.
Printer not discoveredDiscovery protocol blocked or different subnet.Confirm both devices on same network.
