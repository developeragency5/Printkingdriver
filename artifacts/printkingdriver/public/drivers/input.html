Driver Reference

Input Driver (Touchpad / Keyboard)

Translates input gestures and keystrokes.

What Is It?

Input drivers convert physical actions on a keyboard, touchpad, or pointing device into events the operating system can deliver to applications. They handle layout, gestures, and accessibility behaviour.

How It Works

Each key press or finger movement generates a hardware signal. The input driver decodes the signal, applies layout and gesture rules, and emits a standard input event. The OS routes that event to the focused application window.

Key Functions

  • Interprets keystrokes according to the active keyboard layout.
  • Recognises multi-finger gestures on touchpads.
  • Provides palm rejection and adjustable sensitivity.
  • Supports accessibility features like sticky keys.

Why It Matters

Input drivers shape the feel of typing and navigation. Even small bugs here cause missed keystrokes, ghost cursor movement, or unresponsive gestures.

Best Practices

A short checklist to keep this driver healthy and reduce the chance of running into the issues above.

  • Use the OEM Precision Touchpad driver on laptops — it enables modern gestures Windows expects.
  • Install the brand-specific software (Logitech G HUB, Razer Synapse, etc.) for tunable mice and keyboards.
  • Keep firmware on wireless peripherals up to date; the driver alone cannot fix on-device bugs.
  • For gaming, prefer XInput-compatible controllers on Windows for plug-and-play behaviour.
  • Recheck hotkey drivers after major Windows feature updates — they are often replaced.

Most keyboards work via the generic HID driver. A vendor driver is only required for macros, RGB lighting, custom keymaps and high polling-rate features.

Often the manufacturer software / driver has not been installed, so the mouse runs at the default 125 Hz / 800 DPI fallback. Installing it unlocks the full sensor range.

XInput is Microsoft’s modern controller API designed for Xbox-style gamepads. DirectInput is the older API still used by some racing wheels and flight sticks. Windows includes both, but the controller’s driver decides which it presents.
