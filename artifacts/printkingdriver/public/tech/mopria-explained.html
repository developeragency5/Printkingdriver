Print Technology

Mopria Explained

A clear explanation of mopria explained for non-engineers.

Overview

Mopria is the Android equivalent of AirPrint — a vendor-neutral standard that lets any Mopria-certified printer accept print jobs from any Mopria-enabled device without a vendor driver.

What Mopria Is

Mopria is a standard maintained by the Mopria Alliance, a consortium of printer manufacturers. Like AirPrint, it sits on top of IPP-Everywhere. A Mopria-certified printer advertises itself on the local network and accepts print jobs in the standard IPP format.

Where It Runs

Mopria Print Service is built into Android 8 and later. On older versions you can install the Mopria Print Service app from the Play Store. Chrome OS uses native CUPS that includes Mopria support.

How to Check Support

Look for the Mopria logo on the printer box or vendor spec sheet. Mopria publishes the certified-printer list at mopria.org. Most printers from the last seven years are certified.

Mopria vs AirPrint

The two standards do almost the same thing, both built on IPP-Everywhere. AirPrint targets Apple devices, Mopria targets Android. Most modern printers are certified for both — the same hardware accepts jobs from either standard.

Limitations

Like AirPrint, Mopria covers basic print only. Vendor-specific features need the vendor Android app.

Bottom Line

On Android, Mopria is the no-fuss way to print. If you are choosing a printer for a mixed Android/Apple household, look for one certified for both Mopria and AirPrint.
